Projects

StudyLock

StudyLock is an application allowing for young students to focus on their work without distractions. It's a web-based app that uses the webcam to detect if the user is on their phone, and gives them verbal encouragement.

It was created using React for frontend and Flask for backend. The app was created in 24 hours for the WinHacks 2025 hackathon.

Lone Wolf

Lone Wolf is a game where you play as a wolf in the wild. You must hunt for food, communicate with other animals, and explore.

This game was made in 24h for the MasseyHacks X hackathon. It was created using Unreal Engine for game-making, Blender for modeling, and Procreate for visual design.

CHaTOS

CHaTOS is a web application that allows you to chat with an LLM, as if it were the character GLaDOS from the Portal series. It also used text-to-speech to emulate the voice of GLaDOS.

This project was created in 24 hours for the MasseyHacks IX hackathon. It was created using Node.js for backend and vanilla HTML/CSS/JS for frontend.
