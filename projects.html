My Devpost

Lone Wolf

Lone Wolf is a game that was made for the MasseyHacks X hackathon. The game is about a wolf who got lost, and must find his way back to his pack.

CHaTOS

CHaTOS is a program made for the MasseyHacks IX hackathon. The purpose of CHaTOS is to allow someone to talk to characters, fictional or not. This project did not succeed as we wanted it to, but it was worth a shot.

CodeBow

CodeBow was a game that was made for the STEMist Hacks II hackathon. It was intended to be a first-person bow combat game, but most of the game elements had been broken by release and so the original goal had not been completed. It was fun to work with teammates, and this was also my first game-related development endeavour.
